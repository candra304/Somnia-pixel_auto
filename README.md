# SAT SET Pixel Bot

Bot otomatis untuk mewarnai pixel di kontrak Somnia.

## Fitur
- Multi akun (`akun.txt`)
- Dukungan proxy (`proxy.txt`)
- Gas fee adaptif
- Deteksi IP publik proxy
- CLI interaktif

## Cara Pakai
1. `npm install`
2. Siapkan `akun.txt` & `proxy.txt`
3. Jalankan: `node index.js`
